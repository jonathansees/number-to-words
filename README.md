### Run locally

1. Clone master branch of repo
  * `git clone https://github.com/jonathansees/number-to-words.git`

2. Cd into repository
  * `cd number-to-words`

3. Install dependencies
  * `yarn`

4. Run project
  * `yarn start`

### Run tests

`yarn test`

### Deploy

Switch to gh-pages branch and run

`yarn deploy`

### Create a build

`yarn build`
